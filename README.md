**MVB Kombat** is a game made by our team of 6 for Bristol CSS' 2025 GameJam. 

The theme of this GameJam was 'retro' and we have produced a retro-styled fighter game based in Bristol. This game was awarded 1st place among 15 other teams.

## Dependencies

The game relies on the following python libraries:
 - pygame (1.9.6)
 - pygame-widgets (1.0.0)
 - numpy (2.3.5)
